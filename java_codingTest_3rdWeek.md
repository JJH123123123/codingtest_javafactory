problem 40:   

- 문제 이해 → 풀이 방향 : 구현, 연산 3개를 구현하는 부분이 까다로울 수도 있을 거 같다.
- 사용한 자료구조 & 시간복잡도 : O(N)
- (option 참고 문제 & 참고 링크 등등) :  유사한? 문제 백준 :  https://www.acmicpc.net/problem/1942
